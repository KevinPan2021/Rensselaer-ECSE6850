How to run:
1) Download all files, (including "youtube_action_train_data_part1.pkl" and "youtube_action_train_data_part2.pkl")
   and place them a the same directory
2) Run "run_file.py"	

In main() function:
1) Change "saving" to True to train the model and plot
   Change "saving" to False to load the model and run TA testset
2) Change "youtube_action_train_data_part1.pkl" to whatever .pkl
   Model will be loaded and model preformance will be printed
3) Uncomment confusion_plot
